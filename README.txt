Usage: "java -jar TileGame.jar map.txt." The game's argument is the file from which the map is loaded.

Move using the arrow keys. Type "take <itemName>" to pick up an item at a given location; "drop <itemname>" works in the same fashion. Type "inventory" to get a listing of the items that are currently carried.

Images in MapPics are provided by Brian Durney at http://universe.tc.uvu.edu/cs3250/project/project6/project6a.html
